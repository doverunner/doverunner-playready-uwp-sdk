GetSubTitleUrl
Returns a subtitle list if the DASH content contains subtitles. If there is no subtitle, the count will be zero in the list returned. In a UWP environment, subtitles contained in DASH content are not displayed. (The content of mkv, mp4, wmv is displayed by the UWP player.) In this case, you must enter the subtitles of MediaElement(MediaSource, MediaPlaybackItem) before calling MediaElement.source. The subtitle button is generated next to the UWP Player Volume Control button.
InitializeAsync
Initialize PallyCon Proxy Server Specifies the folder where the content was downloaded (on a removable disk). The default folder location is `Windows.Storage.ApplicationData.Current.LocalFolder`. You must have Removable Disk permissions to avoid errors.
