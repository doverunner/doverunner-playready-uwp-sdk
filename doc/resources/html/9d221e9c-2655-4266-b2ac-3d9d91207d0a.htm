Version 1.2

Released on May 2018

Changes in This Release

  • Add DASH content download delegate

    DASH content downloads are provided by SDK, but can be controlled outside the SDK.

    In this case, you can implement the following delegate and notify the handler with CreateDownloadTask ().

    Copy
    public delegate Boolean DownloadSendRequest(String srcUrl, StorageFolder downloadFolder, String destPath)

  • Add DASH Content Multilanguage Support

    DASH Contents has been added to support Multilanguage(subtilte/audio).

    Copy
    public async Task<List<string>> GetSubTitleUrl(Uri mpdUri, string contentName)

    GetSubTitleUrl() is a function PalyConPRSDK.CreateProxyServer that generates and calls a PalyConProxyServer instance.

  • Add to download Multilanguage files

    When downloading content, you have added the ability to download subtitles and audio if it is Multilanguage content.

    You must also have subtitles and audio files when downloading and playing DASH content.

    Subtitles download all languages, and audio downloads high Bitrate of all languages.
