PallyConPRSDK
A class that can create DownloadTask and acquire PlayReady license. To use PallyConPRSDK, you need to set SiteID, SiteKey through Initialize function. You also need to set the MediaProtectionManager in the media player through the CreateProtectionManager function. And you can get the PallyConDownloadTask object through the createDownloadTask method to download the content.
